# TeachC

##位运算器
###使用软件
使用VC6.0 MFC编写
###功能
####1、通过点击图标进行准确的原码、反码、补码计算
![这里写图片描述](https://github.com/will4906/TeachC/blob/master/images/left_part.png)
####2、通过选择右边三排灯泡并通过左边灯泡进行选值，右下边计算符计算出位运算结果
![这里写图片描述](https://github.com/will4906/TeachC/blob/master/images/right_part.png)
###作用
####1、针对数据传输协议的规则进行简单的示范
####2、使使用者更加方便、简单、形象地理解计算机对数据的计算原理
